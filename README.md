# How to run
Enter ```cmd /k code .``` in the folder directory.<br>
Open two new terminals.<br>
- For the first one, type ```air```.
- For the second one, type ```cd client``` type ```npm run dev```, and CTRL + click the link.<br>
Open Postman application and insert the link ```http://localhost:4000/api/todos``` as the base link.<br>
- GET for get all datas.
- POST to add a data.
- PATCH to change the data status.
- DELETE to delete a data.
